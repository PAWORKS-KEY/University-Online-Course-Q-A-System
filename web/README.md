# 如何运行本程序？
## 首先确保你已安装了：
* **Maven**
* **Node.js**
* **Mysql**
* **仓库中自带了npm**

## 运行步骤
1) clone本仓库并进入项目根目录  

2) 准备数据库    
- 执行 `docs/init_database.sql`：  
```bash
mysql -u root -p < docs/initialize_database.sql
```

3) 启动后端（Spring Boot）  
```bash
cd backend/demo
mvn clean package -DskipTests
mvn spring-boot:run
```
- 然后默认监听 `http://localhost:8080`。

4) 启动前端（Vue）  
```bash
cd frontend
npm install   # 如果首次运行
npm run dev
```
- 默认地址 `http://localhost:5173`

5) 项目部署完成，开始使用



# 📚 在线学习资源共享与答疑互动平台项目介绍

## 🚀 项目简介

本项目旨在搭建一个高效、便捷的**大学生在线学习资源共享与答疑互动平台**，以提高学生的在线学习效率。平台通过整合学习资源管理、互动问答系统和多角色权限控制，构建一个教师、学生、管理员协同的学习生态系统。

## ✨ 核心功能亮点

* **资源共享：** 支持学生和教师上传、下载各类学习资料。
* **互动问答：** 实时提问、教师专业答疑，并提供新回答通知提醒。
* **多角色权限：** 精细化区分管理员、教师和学生功能，确保平台高效运营和内容质量。
* **内容检索：** 强大的关键字、课程、教师等多维度搜索功能。

## 👥 系统角色说明

| 角色 | 权限说明 | 账号创建方式 |
| :--- | :--- | :--- |
| **管理员** | 平台最高权限，负责课程、教师、内容和系统维护。 | 系统初始化创建。 |
| **学生** | 自由注册，参与资源共享与学习问答。 | 自由注册。 |
| **教师** | 负责答疑和上传高质量资源，拥有课程内容权限。 | 由管理员添加。 |

---

## 🔧 功能模块详情

### I. 管理员功能模块

管理员拥有平台内容的最高管理权限，以保证平台的健康运行。

| 模块 | 功能描述 | 详细操作 |
| :--- | :--- | :--- |
| **课程管理** | 管理平台开设的所有课程信息。 | **增、删、改** 课程信息（名称、授课教师、内容描述、开课学院）。 |
| **教师管理** | 管理教师账号信息及其所教授的课程。 | **增、删、改** 教师信息；设置教师讲授课程（可多门）、简介、职称。 |
| **学习资源管理**| 维护资源内容的合规性与准确性。 | **删除** 不合规或违规的资源与资料；**修改** 资源说明信息。 |
| **问答内容管理**| 维护问答区内容的质量与秩序。 | **删除、修改** 任意学生提问、教师回答；管控错误或违规内容。 |

### II. 学生功能模块

学生是平台的主要使用者，可参与资源共享和学习互动。

| 模块 | 功能描述 | 详细操作 |
| :--- | :--- | :--- |
| **通知提醒** | 接收问答互动反馈。 | 若发布的问题被教师回答，系统提示：“您有 **X** 条新回答”，可点击查看。 |
| **学习资源浏览**| 查找和浏览所需学习资料。 | 按**课程分类**浏览所有资源；支持**关键字搜索**（标题 / 内容）。 |
| **上传学习资料**| 共享优质学习资源。 | 上传资源文件（如 PDF、图片、压缩包），并填写：**资源标题、简介、所属课程**。 |
| **上传时间**：系统自动记录。| |
| **查看资源详情** | 了解资源信息和受欢迎程度。 | 显示：上传者、资源简介、**下载次数**。 |
| **学生问答区** | 提出学习中遇到的问题。 | 进入课程页面**提出问题**（提问标题、内容、时间）；支持**上传图片作为附件**；支持分页显示问题列表。 |
| **提问搜索** | 快速定位相关问答内容。 | 按**学科、教师、关键字**检索全站问答。 |
| **个人中心** | 管理自己的贡献内容。 | **查、改、删** 自己上传的学习资料；**查、改、删** 自己提出的问题；查看已被回答的问题及回答内容。 |

### III. 教师功能模块

教师是平台答疑和资源质量的保障者。

| 模块 | 功能描述 | 详细操作 |
| :--- | :--- | :--- |
| **新问题提醒** | 提醒教师及时进行答疑。 | 显示：“您所上课程有 **X** 条未回答问题”，可点击进入答疑页面。 |
| **发布学习资源**| 上传高质量、官方认证的资料。 | 填写：**资源标题、说明内、所属课程**；可上传附件（PDF、图片等）。 |
| **回答学生问题**| 专业解答学生的学习疑问。 | 查看所上课程的全部学生提问；**回答问题**（支持文本 + **图片附件**）；**修改或删除**自己的回答。 |
| **学科权限设置**| 灵活控制资源可见范围。 | 设置：某课程学习资源**仅对本班学生可见** 或 **全部学生可见**。 |

### IV. 通用功能模块

所有用户均可使用的基本账户管理功能。

| 模块 | 功能描述 |
| :--- | :--- |
| **账户安全** | 修改个人登录密码。 |
| **个人资料管理**| 修改个人资料（头像、邮箱等）。 |

## 🛠️ 技术栈选择

为了实现以上功能，使用以下技术栈：

| 层面 | 推荐技术 | 备注 |
| :--- | :--- | :--- |
| **前端 (Frontend)** |  Vue | 用于构建动态、响应式的用户界面。 |
| **后端 (Backend)** | Spring Boot | 提供 RESTful API 接口和业务逻辑支持。 |
| **数据库 (Database)** | MySQL | 存储用户、课程、资源和问答等结构化数据。 |
